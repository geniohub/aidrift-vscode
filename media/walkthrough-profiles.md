# Multiple hosts, multiple users

AI Drift profiles let you switch between different hosts and accounts without re-entering credentials.

- **default** — `https://drift.geniohub.com` (preconfigured)
- Add your own — e.g. `dev` pointing at `http://localhost:3331/api`

Switch at any time from the status bar (`aidrift: <profile>`) or via **Drift: Switch Profile**.

Each profile stores its own token in your OS keychain, so signing into one doesn't disturb the other.
