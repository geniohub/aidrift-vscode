# Sign in to AI Drift

AI Drift watches your Claude Code and Codex conversations and warns you when a session is drifting off-task.

The default host is **https://drift.geniohub.com**.

Clicking **Sign in via browser** opens that page, you sign in (or sign up), and the editor picks up the token automatically.

Prefer to stay offline? Run the CLI — `drift login --token …` — or use **Drift: Sign In with Token** to paste a Personal Access Token manually.
